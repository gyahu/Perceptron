The following folder contains:
- perceptron.h and perceptron.cpp: The files required to use an implementation of a perceptron
- example.cpp: An example of the use of the perceptron, modeling the NAND, AND and OR gate, and the summing bit gate.
- Makefile: A file to compile the code and being able to run it by calling the output.out file

All of these files can be found on gyahu's github.
